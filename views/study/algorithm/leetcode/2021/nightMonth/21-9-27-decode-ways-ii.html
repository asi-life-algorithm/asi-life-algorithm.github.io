    【算法】 leetcode-9-27号每日一题——639. 解码方法 II

    差不多的贝壳 2021-9-27 动态规划每日一题

    # 题目描述

    题目链接 (opens new window)

    一条包含字母 A-Z 的消息通过以下的方式进行了编码：

    'A' -> 1
'B' -> 2
...
'Z' -> 26

    要 解码 一条已编码的消息，所有的数字都必须分组，然后按原来的编码方案反向映射回字母（可能存在多种方式）。例如，"11106" 可以映射为：

    "AAJF" 对应分组 (1 1 10 6) "KJF" 对应分组 (11 10 6) 注意，像 (1 11 06) 这样的分组是无效的，因为 "06" 不可以映射为 'F' ，因为 "6" 与 "06" 不同。

    除了 上面描述的数字字母映射方案，编码消息中可能包含 '' 字符，可以表示从 '1' 到 '9' 的任一数字（不包括 '0'）。例如，编码字符串 "1" 可以表示 "11"、"12"、"13"、"14"、"15"、"16"、"17"、"18" 或 "19" 中的任意一条消息。对 "1*" 进行解码，相当于解码该字符串可以表示的任何编码消息。

    给你一个字符串 s ，由数字和 '*' 字符组成，返回 解码 该字符串的方法 数目 。

    由于答案数目可能非常大，返回对 10^9 + 7 取余 的结果。

    # 示例

    输入：s = "*"
输出：9
解释：这一条编码消息可以表示 "1"、"2"、"3"、"4"、"5"、"6"、"7"、"8" 或 "9" 中的任意一条。
可以分别解码成字符串 "A"、"B"、"C"、"D"、"E"、"F"、"G"、"H" 和 "I" 。
因此，"*" 总共有 9 种解码方法。

    输入：s = "1*"
输出：18
解释：这一条编码消息可以表示 "11"、"12"、"13"、"14"、"15"、"16"、"17"、"18" 或 "19" 中的任意一条。
每种消息都可以由 2 种方法解码（例如，"11" 可以解码成 "AA" 或 "K"）。
因此，"1*" 共有 9 * 2 = 18 种解码方法。

    输入：s = "2*"
输出：15
解释：这一条编码消息可以表示 "21"、"22"、"23"、"24"、"25"、"26"、"27"、"28" 或 "29" 中的任意一条。
"21"、"22"、"23"、"24"、"25" 和 "26" 由 2 种解码方法，但 "27"、"28" 和 "29" 仅有 1 种解码方法。
因此，"2*" 共有 (6 * 2) + (3 * 1) = 12 + 3 = 15 种解码方法。

    提示：

    1 <= s.length <= 10^5 s[i] 是 0 - 9 中的一位数字或字符 '*'

    # 题解

    动态规划: dp[n]表示从0-i的位置的解法次数。

    dp[i] 主要是根据 dp[i - 1]和 dp[i - 2]得来的。

    主要规则如下：

    如果 s[i] 为0，如果s[i - 1]为0，3-9则不符合规矩直接返回0.否则dp[i] = dp[i - 1];

    如果 s[i] 为数字，如果s[i + 1]为0，则i 和 i + 1一定要绑定在一起。

如果s[i - 1]为 * 号，则 * 号如何和 i绑定，则 * 号只能为1 或者 2(当前的数字要小于等于6)，dp[i] = dp[i - 1] + dp[i - 2] + (dp[i - 2](如果s[i] 小于等于6时))。

如果s[i - 1]为数字，如果s[i - 1]为1或者 s[i - 1]为2且s[i]小于等于6时，dp[i] = dp[i - 1] + dp[i - 2]。否则dp[i] = dp[i - 1];

    如果s[i] 为*号, 如果s[i + 1]为0，则i 和 i + 1一定要绑定在一起。则 * 号只能为 1 和 2. dp[i] = dp[i - 2] * 2;

如果s[i - 1]为* 号，则有两种情况，两个星号绑定在一起，则为[11 - 19],[21-26],总共15个；两个星号分开，则当前*号表示[1-9].即 dp[i] = dp[i - 2] * 15 + dp[i - 1] * 9;

如果s[i - 1]为1，则有两种情况，s[i - 1]和*号绑定在一起，则*号为[1-9], 如果s[i - 1]分开，* 号也为[1-9]；则 dp[i] = dp[i - 2] * 9 + dp[i - 1] * 9

如果s[i - 1]为2，则有两种情况，s[i - 1]和 *号绑定在一起，则*号为[1-6], 如果s[i - 1]分开，* 号为[1-9]; 则 dp[i] = dp[i - 2]* 6 + dp[i - 1]*9

否则*号只表示[1-9],dp[i] = dp[i - 1] * 9

        public int numDecodings(String s) {
        long[] dp = new long[s.length() + 1];
        dp[0] = 1;
        int mod = 1000000007;
        char[] sChar = s.toCharArray();
        if (sChar[0] == '*') {
            if (sChar.length != 1 && sChar[1] == '0') {
                dp[1] = 2;
            } else {
                dp[1] = 9;
            }
        } else if (sChar[0] == '0'){
            return 0;
        } else {
            dp[1] = 1;
        }
        for (int i = 1; i < sChar.length; i++) {
            if (sChar[i] == '0') {
                if ((sChar[i - 1] <= '9' && sChar[i - 1] >= '3') || (sChar[i - 1] == '0')) {
                    return 0;
                }
                dp[i + 1] = dp[i];
                dp[i + 1] = dp[i + 1] % mod;
                continue;
            }
            if (sChar[i] == '*') {
                if (i != sChar.length - 1 && sChar[i + 1] == '0') {
                    dp[i + 1] = dp[i] * 2;
                } else if (sChar[i - 1] == '*') {
                    dp[i + 1] = dp[i - 1] * 15 + dp[i] * 9;
                } else if ('1' == sChar[i - 1]) {
                    dp[i + 1] = dp[i - 1] * 9 + dp[i] * 9;
                } else if (sChar[i - 1] == '2') {
                    dp[i + 1] = dp[i - 1] * 6 + dp[i] * 9;
                } else {
                    dp[i + 1] = dp[i] * 9;
                }
            } else {
                if (i != sChar.length - 1 && sChar[i + 1] == '0') {
                    dp[i + 1] = dp[i];
                } else if (sChar[i - 1] == '*') {
                    dp[i + 1] = dp[i] + dp[i - 1];
                    if (sChar[i] <= '6') {
                        dp[i + 1] = dp[i + 1] + dp[i - 1] ;
                    }
                } else if ('1' == sChar[i - 1] || ('2' == sChar[i - 1] && sChar[i] <= '6')) {
                    dp[i + 1] = dp[i] + dp[i - 1];
                } else {
                    dp[i + 1] = dp[i];
                }
            } 
            dp[i + 1] = dp[i + 1] % mod;
        }
        return (int)dp[sChar.length];
    }
