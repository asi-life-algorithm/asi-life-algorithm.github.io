    【算法总结】 01背包算法问题总结

    差不多的贝壳 2021-6-12 01背包

    最近leetcode的每日一题全是01背包问题。

    所以以此来做个总结。

    01背包是动态规划中的一个小部分。

    其核心是 n个背包，每个背包的有一定容量wi，问往n个背包装多个重量为vi和价值mi的石头，问最终能达到mi最大价值为多大。

    2021-06-06的题目1和0， 可以理解为：有一个背包，容量m个1，n个0，往这个这一个背包装m1个1和n1个0的石头，问最大能装多少个石头。

    2021-06-07的题目目标和，可以理解为：有一个背包，容量为target，往这个背包装m个正数和n个负数，问能有多少种装法。

    2021-06-08的题目最后一块石头的重量 II，可以理解为，有一个背包，往这个背包里面装m个整数和n个负数，问最小的容量是多少。

    2021-06-09的题目盈利工作，可以理解为，有profit.length个背包，每个背包至少需要group[i]重量的石头，才可以获得profit[i]的价值，总共有n个石头，问有多少种装法。

    2021-06-10的题目零钱兑换，可以理解为，有一个背包，容量为amount，往这个背包装x个重量为石头，问达到amount时的重量有多少种装法。

    2021-06-11的题目完全平方数，可以理解为，有一个背包，容量为n，往这个背包装重量为完全平方的石头，问达到n使用的最少石头。

    2021-06-12的题目数位成本和为目标值的最大数字，可以理解为：有cost.length种背包，每个背包有n种，总容量为target，每种类型的背包重cost[i]，问当把所有的背包重量达到容量target时，问如何使背包个数最多，且数量最大。

    01背包的主要核心就是：该石头是否存放到到当前背包，0为不存放，1为存放。

    所以常常定义数组dp[n][target],表示第n个背包，容量为target时的价值。

    对于第i块石头，第j个背包，当前容量为v，i个是否有两个方案：

    1. 第j个背包不存放第i块石头，则dp[j][v] = 只能等于dp[j - 1][v]

    2. 第j个背包存放第i块石头，重量为wi，则v肯定得大于wi，因为不大于wi，则第i块石头肯定放不下所以dp[j][v] = Math.max(dp[j - 1][v], dp[j - 1][v - wi]);所以第j个背包达到容量为v时的价值有两种：第一种为j-1个背包达到v重量时的价值和第j-1个背包达到v-wi的重量的价值。

    
// 01 背包问题的核心模板，但是并不一定和套用所有的题目。但是理解题目后在使用此模板进行解答。
//N为物品数量
//target为背包总容量
//weights[]为每个物品的体积
//values[]为每个物品的价值
 
public int zeroOneKnapsack(int N, int target, int[] weights, int[] values){
    int[][] dp = new int[N+1][target+1];
 
    for(int i = 1; i <= N; i++){
        //每个石头的体积和价值
        int w = weights[i-1], v = values[i-1];
        for(int j = 1; j <= target; j++){
            if(j >= w){
                // 可存放当前石头
                dp[i][j] = Math.max(dp[i-1][j], dp[i-1][j-w]+v);
             }else{
                // 不可存放
                dp[i][j] = dp[i-1][j];
            }
        }
    }
    return dp[N][target];
}

// 优化，dp空间优化
public int zeroOneKnapsack(int N, int target, int[] weights, int[] values){
    int[] dp = new int[target+1];
 
    for(int i = 1; i <= N; i++){
        //每个石头的体积和价值
        int w = weights[i-1], v = values[i-1];
        for(int j = 1; j <= target; j++){
            if(j >= w){
                // 可存放当前石头
                dp[j] = Math.max(dp[j], dp[j-w]+v);
             }else{
                // 不可存放
                dp[j] = dp[j];
            }
        }
    }
    return dp[target];
}
